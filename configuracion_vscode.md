# Configuración de vscode

### Extensiones mas comunes

|Extensión|Descripción breve|
|---|---|
|Auto Rename Tag|Renombra automáticamente etiquetas HTML/XML emparejadas.|
|Better Comments|Resalta comentarios con colores según su propósito (TODO, WARNING, etc.).|
|Code Runner|Ejecuta fragmentos de código directamente desde VS Code.|
|Excel Viewer|Permite abrir y visualizar archivos Excel dentro del editor.|
|GitLens|Amplía Git con historial, blame y comparaciones avanzadas.|
|Houston|Herramienta para gestión y visualización de entornos remotos (según configuración).|
|Indent-Rainbow|Colorea niveles de indentación para mejorar legibilidad.|
|Julia|Soporte completo para el lenguaje Julia.|
|Jupyter|Permite trabajar con notebooks Jupyter dentro de VS Code.|
|Live Server|Lanza un servidor local con recarga automática para desarrollo web.|
|Polacode|Genera imágenes estilizadas de fragmentos de código.|
|Prettier|Formateador automático de código según estándares definidos.|
|Python|Soporte oficial para Python: linting, debugging, testing.|
|Rainbow CSV|Colorea columnas CSV para facilitar su lectura.|
|SQLite Viewer|Permite abrir y explorar bases de datos SQLite.|
|Thunder Client|Cliente HTTP integrado para probar APIs REST.|
|vscode-icons|Añade iconos específicos según tipo de archivo y carpeta.|
|vscode-pdf|Permite abrir y leer archivos PDF dentro de VS Code.|
